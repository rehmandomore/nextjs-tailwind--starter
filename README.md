# Nextjs Typescritp TailwindCSS 3

```bash

yarn create next-app <app-name> --typescript


# https://tailwindcss.com/docs/guides/nextjs

yarn add -D tailwindcss postcss autoprefixer

yarn tailwindcss init -p

yarn add -D @tailwindcss/typography @tailwindcss/forms @tailwindcss/line-clamp @tailwindcss/aspect-ratio tailwindcss-debug-screens
```
